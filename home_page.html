About the Platform

This platform integrates an AI-powered E-Commerce Chatbot with an automated Documentation Sync Agent. It ensures real-time user assistance while maintaining up-to-date technical documentation aligned with source code changes.

Documentation Sync Agent (EPIC 2)

The Documentation Sync Agent is an event-driven workflow built using Flowise / Flogentic orchestration. It reacts to repository events and generates accurate technical documentation using AI.

Event Driven

Automatically triggers on commits and pull-request workflows.

Change Detection

Identifies impacted files, modules, and components.

AI Documentation

Generates LLD, summaries, and architecture explanations.

Access Controlled

Documentation shared only with authorized work or school emails.

Public documentation publishing is intentionally disabled to maintain compliance and internal governance standards.
